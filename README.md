## About

This portfolio was developed to showcase my projects, technical skills and interests through a clean, modern and user-focused interface.

Rather than being a simple personal website, the project was designed as a complete frontend application where design, code quality and user experience are equally important.

The interface follows a minimalist approach inspired by products such as Apple and Notion, prioritizing readability, consistency and simplicity.

---

## Features

* Modern and minimalist interface
* Fully responsive layout
* Smooth scrolling navigation
* Interactive project showcase
* Dedicated project pages
* Modular code structure
* Lightweight and fast loading
* SEO-friendly semantic HTML

## Design Philosophy

The project follows a minimalist design philosophy where content takes priority over decoration.

Special attention was given to:

* Visual hierarchy
* Consistent spacing
* Typography
* User experience
* Responsive layouts
* Smooth interactions
* Maintainable code

Every section was designed to keep navigation intuitive while presenting information in a clear and organized way.

Open the project folder:

